Task 1 - JioHotstar Dataset Preprocessing
Objective

To clean and preprocess the JioHotstar dataset for machine learning.

Tools Used
Google Colab
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Steps Performed
Imported the JioHotstar dataset.
Explored dataset structure and checked data types.
Identified and handled missing values.
Encoded categorical features into numerical format.
Standardized numerical columns using feature scaling.
Visualized outliers using boxplots.
Removed outliers using the IQR (Interquartile Range) method.
Generated a correlation heatmap to analyze feature relationships.
Saved the cleaned and preprocessed dataset for machine learning.
Dataset

JioHotstar Movies and Shows Dataset from Kaggle.
